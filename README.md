# cs

## 前端开发技术栈 (Frontend Development Stack)

### 跨平台框架 (Cross-Platform Frameworks)

| 框架 (Framework) | 核心语言 (Core Language) | 描述 (Description) |
|-----------------|-------------------------|-------------------|
| **React Native** | JavaScript / TypeScript | Facebook 开发的跨平台移动应用框架 |
| **Flutter** | Dart | Google 开发的跨平台 UI 框架 |

### 支持平台 (Supported Platforms)

| 平台 (Platform) | React Native | Flutter |
|----------------|--------------|---------|
| iOS App (苹果端) | ✅ | ✅ |
| Android App | ✅ | ✅ |
| H5 (Web) | ✅ (React Native Web) | ✅ (Flutter Web) |
| 小程序 (Mini Program) | ✅ (需配合 Taro/uni-app) | ✅ (需配合 flutter_wechat/mxflutter) |

### 开发语言详情 (Language Details)

#### React Native
- **主要语言**: JavaScript / TypeScript
- **UI 组件**: React 组件化开发
- **状态管理**: Redux, MobX, Zustand 等
- **适用场景**: 需要快速开发、团队熟悉 React 生态

#### Flutter
- **主要语言**: Dart
- **UI 组件**: Widget 组件化开发
- **状态管理**: Provider, Riverpod, Bloc 等
- **适用场景**: 需要高性能 UI、自定义动画丰富的应用
