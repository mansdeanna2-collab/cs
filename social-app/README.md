# 影视 - 视频播放平台

一个移动端视频播放应用，采用React + TypeScript构建。

## 功能特点

- 🎬 **首页** - 视频推荐、分类浏览、轮播图
- 🌐 **暗网** - 私密内容专区  
- 📺 **直播** - 直播房间列表
- 🎮 **游戏** - 游戏中心
- 👤 **我的** - 个人中心

## 界面特点

- 深色主题设计
- 5个底部导航标签
- 视频分类标签（推荐、国产、日本、动漫、福利）
- 每个分类展示5个视频（1大4小布局）
- 换一换和查看更多操作

## 可用脚本

### `npm start`

启动开发模式，打开 [http://localhost:3000](http://localhost:3000) 在浏览器中查看。

### `npm test`

运行测试。

### `npm run build`

构建生产版本到 `build` 文件夹。
